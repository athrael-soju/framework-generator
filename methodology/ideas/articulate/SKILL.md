# Articulate

Frame findings for maximum impact. Create deliverables.

## Inputs

| Input | Source |
|-------|--------|
| identity_profile | Identity stage |
| evaluation | Evaluate stage |
| contract | Deliverable requirements |

## Process

**1. Position** — Define contribution (novel, confirmation, framework)
**2. Target** — Adapt for audience
**3. Recommend** — Develop actionable guidance
**4. Roadmap** — Create implementation plan

## Audience Adaptation

| Audience | Format | Depth |
|----------|--------|-------|
| Executive | Summary + recs | High-level |
| Technical | Full report | Deep |
| Marketing | Slides + points | Medium |

## Output Format

```markdown
# Deliverable: [Title] | YYYY-MM-DD

## Executive Summary
[2-3 sentences: what, so what, now what]

## Key Findings
1. Finding 1 — implication
2. Finding 2 — implication

## Recommendations
| Priority | Recommendation | Effort | Impact |
|----------|----------------|--------|--------|
| 1 | Do X | Low/Med/High | High |

## Roadmap
| Phase | Actions | Timeline |
|-------|---------|----------|
| 1 | Action | Timeframe |

## Appendix
[Supporting detail]
```

## Quality Criteria

- [ ] Findings framed for audience
- [ ] Claims supported by evidence
- [ ] Recommendations actionable
- [ ] Limitations acknowledged

## Completion

Present: deliverable draft, format, recommendations. Approve → Share.
