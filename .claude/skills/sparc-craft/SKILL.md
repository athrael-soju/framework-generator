# SPARC Craft Stage

Develop tailored outreach based on analysis.

## Activities

| Activity | Inputs | Outputs |
|----------|--------|---------|
| Message Drafting | Prospect Analysis, templates, buyer profile | Outreach variants |
| Brief Preparation | Analysis, likely questions | Supporting Brief |

## Outreach Structure

| Component | Purpose |
|-----------|---------|
| Hook | Reference specific signal or content |
| Insight | Non-obvious observation from research |
| Bridge | Connect their gap to your capability |
| Ask | Specific, low-commitment next step |

## Channel Preference Order

1. Warm intro
2. Content engagement then DM
3. Cold DM
4. Email

## Skills to Execute

1. **decision_maker_profile** - Build profile of target buyer
2. **outreach_draft** - Draft personalized message with variants

## Message Template

```
[Hook - reference their specific signal/content]

[Insight - non-obvious observation demonstrating research]

[Bridge - connect their gap to your expertise]

[Ask - specific, low-friction next step]
```

## Output

1. **Outreach Message** - Primary message ready to send
2. **Variants** - 2-3 alternative versions with different hooks
3. **Supporting Brief** - Talking points for likely responses

## Quality Criteria

- [ ] Hook references specific, recent signal
- [ ] Insight demonstrates research depth
- [ ] Ask is concrete and low-friction
- [ ] Brief prepared for likely responses
